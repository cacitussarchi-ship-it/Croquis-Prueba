# Feria del Mueble Sarchí 2026

App de selección de stands — CACITUS · Cámara de Comercio

## Deploy en Vercel

1. Suba esta carpeta a un repositorio de GitHub
2. Vaya a vercel.com → New Project → importe el repo
3. En "Framework Preset" seleccione **Other**
4. Haga clic en Deploy

O use Netlify Drop: arrastre la carpeta a netlify.com/drop
